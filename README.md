# VeggiePick（野菜の収穫体験予約サイト）

## サービス概要
家族や友人グループが手軽に「野菜の収穫体験」を予約できるサイトです。  
利用者は空き日程を確認して予約ができ、農園側は予約状況を一元管理できます。  
季節イベントとして人気の芋掘りを、もっと身近で簡単に楽しめるようにします。

## このサービスへの思い・作りたい理由
父が芋掘り体験を行っている農園の手伝いをしており、予約をすべて電話で受け付けているため対応が大変だと聞きました。  
「この課題をシステムで解決してあげたい」という思いから開発を決意しました。  
また、予約がオンラインで簡単にできるようになれば、利用者にとっても参加のハードルが下がり、農園の利用者増加にもつながると考えています。

## 想定ユーザー
### メインターゲット
- **一般利用者（家族・カップル・友人グループ）**
  - 手軽に自然体験を楽しみたい層
  - オンラインでスムーズに予約したいニーズがある

### サブターゲット  
- **農園スタッフ・管理者**
  - 予約の管理や人数調整を効率化したい
  - 電話対応の負担を減らしたい

## サービス利用イメージ
### 一般利用者（家族・カップル・友人グループ）
  -	営業日カレンダーから空き状況を確認し、希望日をフォーム入力し簡単に予約
	-	自動返信メールで内容を確認し、当日は予約番号を提示して収穫体験を楽しむ

### 農園スタッフ・管理者
  -	管理者画面にログインし、予約一覧から日付や人数を確認・管理
	-	電話対応の負担を減らし、当日の運営に集中できる

## サービスの差別化ポイント・推しポイント
- **シンプルで直感的なUI** - スマホでも予約しやすい画面設計
- **地域特化型** - 地元の収穫体験に焦点を当て、全国展開サイトとの差別化
- **運営側の負担軽減** - 予約枠の自動制御・キャンセル管理でミスを削減

「誰でも迷わず使える予約システム」を目指します。

## 機能候補

### MVPリリース時（最小限の価値提供）
- **体験日程の一覧表示** - カレンダー形式で空き状況を視覚的に表示
- **予約フォーム** - 氏名・電話番号・人数・希望日時の入力
- **予約確認・キャンセル機能** - 予約番号による照会とキャンセル
- **管理者ログイン機能** - Deviseによる認証→MVC時はBasic認証
- **管理者向け予約一覧画面** - 日付別・ステータス別の予約管理
- **予約確認メール通知** - 予約完了・キャンセル時の自動メール送信　（本リリース時の実装に変更する可能性あり）

### 本リリース時（体験向上）
- 日程の定員設定と自動締め切り
- 画像付きの体験紹介ページ
- 周辺施設の情報と一日のモデルコース紹介ページ
- 口コミ・レビュー機能

### 将来的な拡張機能
- 多言語対応（日本語・英語）
- 他農園への横展開機能
